Простой REST для работы с книгами

GET /api/books - Получить список книг

GET /api/books/{id} - Получить книгу по id. Если книги не существует ответ со HTTP-статусом 404

POST /api/books [Request Body] - Добавить книгу

Формат [Request Body] - JSON с полями year, author, title,

Пример: {'year': 1924, 'author': 'Franz Kafka', 'title': 'The Castle'}
